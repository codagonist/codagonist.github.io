# codagonist.github.io

Personal website of the codagonist

# HOW TO

- go into the codagonist.github.io folder
- make your changes (create files, folders, etc)
- stage all changes: git add .
- add a git commit message: git commit -m "the commit message"
- push the changes to the repo: git push

The changes should be visible in the repo. It could take a minute or two the website displays the changes.
